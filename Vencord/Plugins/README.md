>```
>`npm install --global corepack@latest`
>```
>```
>`corepack enable pnpm`
>```
>```
>`git clone https://github.com/Vendicated/Vencord && cd Vencord && pnpm install --frozen-lockfile && pnpm build && cd src && mkdir userplugins && cd userplugins && start .`
>```
>```
>`pnpm build && pnpm inject`
>```
